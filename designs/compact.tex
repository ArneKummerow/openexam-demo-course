% Compact design — tight margins, smaller default font, two-column
% friendly. The points badge moves to the margin via
% \oedesignpointsplacement = margin (set by the renderer).

% it with \geometry{...} — a second \usepackage[...]{geometry} would
% raise "Option clash for package geometry" and abort the PDF build.
